<h1>Installation Instructions</h1>
<ul>
  <li>
    <p>Ensure a form of hypervisor is installed on your machine, free options include <a href="https://www.oracle.com/virtualization/virtualbox/" target='_blank' rel='noopener noreferrer'>Oracle VirtualBox,</a> <a href="https://www.vmware.com/products/desktop-hypervisor/workstation-and-fusion" target='_blank' rel='noopener noreferrer'>VMware Workstation Pro,</a> or <a href="https://learn.microsoft.com/en-us/windows-server/virtualization/hyper-v/get-started/install-hyper-v?tabs=powershell&pivots=windows" target='_blank' rel='noopener noreferrer'>Hyper-V (Windows only).</a></p>
    <p>Only one hypervisor should be active at a time</p>
  </li>
  <li>
    <p>Download the virtual machine from the <a href="https://drive.google.com/drive/folders/1kI1M6oUScuLeLakMTziUkeQZGeN7aBah?usp=sharing" target='_blank' rel='noopener noreferrer'>Google Drive</a></p>
    <p>Files within the virtual machine can be viewed in the <a href="https://github.com/diamondhicks22-glitch/OWASPTop10CTFCapstoneVM/releases/tag/Releases" target='_blank' rel='noopener noreferrer'>Github Repository</a></p>
  </li>
  <li>
    Import the virtual machine (normally seen as Import Application option) into the hypervisor and run it.
  </li>
  <li>
    Booting can take around 10-30 seconds for Apache and other services into the Virtual Machine to start.
    After the machine boots, open your local browser and go to http://localhost/.
  </li>
</ul>
